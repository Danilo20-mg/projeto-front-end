Como Usar Seus Cupons

Passo a passo:

  1. Adicione os produtos desejados ao carrinho
  2. Vá para o checkout (finalizar compra)
  3. Procure o campo "Cupom de desconto"
  4. Digite o código do cupom exatamente como aparece
  5. Clique em "Aplicar cupom"
  6. O desconto será aplicado automaticamente
  7. Finalize sua compra com desconto!

Dicas importantes:

  • Cupons são sensíveis a maiúsculas e minúsculas
  • Digite o código exatamente como aparece
  • Verifique sempre a data de validade
  • Confirme se sua compra atende às condições
  • Apenas um cupom por pedido

Regras Gerais

Condições de uso:

  • Válido apenas para compras online
  • Não cumulativo com outras promoções
  • Não aplicável a produtos já com desconto
  • Não é possível trocar por dinheiro
  • Cupons têm data de validade
  • Um cupom por CPF/pedido

Produtos excluídos:

  • Produtos em liquidação
  • Cartões presente
  • Taxas de entrega
  • Produtos de marcas premium (quando especificado)

Em caso de problemas:

  • Verifique se o código está correto
  • Confirme se o cupom ainda é válido
  • Verifique se atende às condições mínimas
  • Entre em contato conosco se persistir
